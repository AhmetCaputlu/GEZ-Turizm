# GEZ-Turizm

Proje .NET 7.0 ve N-Tier katmanlı mimari anlayışıyla geliştirildi. 
OOP ve SOLİD prensiplerine bağlı kalınarak geliştirdiğim bu projede; 
-Entity Framework (ORM),ASP.NET Core MVC,WEB API (RESTful),Microsoft SQL Server,JWT ve Cookie teknolojileri de bulunmaktadır.

Excel tabanlı kayıt sistemi oluşturmuş bir işletmenin modern programlar vasitasıyla işine 
çeviklik ve verimli iş gücü katabilmeyi amaçlayan bu projede temel düşünce; performans odaklı, yüksek güvenlikli, kullanıcı dostu ve gelişime açık bir program sunmaktır.

Projenin Temel Özellikleri

-Repository Pattern : Uygulama tek bir veritabanı bağlantısı ile doğrudan ilişkilendirilmemeli ki yeniden kullanılabilirlik ve gelişime açık olma opsiyonunun önü kapanmasın.Dependency Injection (DI) yöntemi ile bağımlılıklar kontrol edilmiştir.

-SOLİD : Uygulama, başka bir geliştirici ekip tarafından yeni bir hizmet eklenmesi, halihazırda kullanılan bir hizmetin iş mantığının güncellenmesi veya kullanılmayan mevcut bir hizmetin kaldırılması durumlarında kolay aksiyon alabilmesi amacıyla SOLİD prensipleri ön planda tutuldu.

-UI/UX : Kullanıcı arayüzünde renk uyumu ve kullanım kolaylığı sağlandığı gibi kullanıcı basit bir veya birkaç işlem için çok sayıda View dosyası içerisinde kaybolmaz.
